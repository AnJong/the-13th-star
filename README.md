# 13번째 별

가을, 김포의 어느 단독주택. 작은 외계 생명체가 떨어진다.
인터랙티브 단편 데모.

## 라이브

https://anjong.github.io/the-13th-star/

## 기술 스택

- Vanilla HTML / CSS / JavaScript
- GSAP (텍스트 애니메이션)
- Pretendard 폰트
- 모바일 우선 반응형 + 수직 스냅 스크롤

## 로컬 실행

브라우저에서 `index.html` 직접 열거나:

```sh
python3 -m http.server 8000
# http://localhost:8000
```

## 배포

`main` 브랜치에 push하면 GitHub Pages 자동 배포.

## 산문 출처

산문 본문은 형제 폴더 `et/final/novel.md`에서 비트별로 옮겨 `data.js`에 임베드.
